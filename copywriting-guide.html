WARROOM邮件一键生成秘籍：

你是一名资深的联想销售，请参考以下维度，根据提供的信息帮忙给老板写一封专业高效的价格申请邮件。

【项目情况】
  • 投标时间：
  • 投标形式：公开招标 / 竞争性谈判 / 邀请招标 / 企业询价 / 框架续签（初步判断项目竞争烈度）
  • 销售方式：直销/渠道销售 → 需说明留利的合理性
  • 评标方式：综合评分 / 最低价中标
  • 评标细则：最低价满分 / 平均价满分
  • 分包规则：整包 / 每个标段1包 / 其他
  • 中标规则：独家中标 / 多家入围
  • 采购方式：统签分采（1年以上/1年/半年/季度） / 一次性下单
  • 历史情况：需说明客户历史采购情况（包括联想历史下单数量）
【竞争情况】
  • 竞争对手：列出具体竞品品牌（如HP/Dell）
  • 机型对位：明确竞品型号与联想对标机型
  • 项目竞争态势：分析当前竞争激烈程度及优劣势
【价格逻辑】
  • 申请价格逻辑：参考市场信息（如公开招标价、行业均价）
  • 提供对手价格信息（需具体数据支撑）
  • 渠道留利组成：进销差价结构说明（如留利5%的合理性）
【毛利改善】
  • 改善计划：需提供可落实的具体举措（例如：资产回收、增值服务捆绑、后续运维合约等）
